# Poli

Poli is a calm, sensory-friendly productivity app for autistic and ADHD adults.

## What is included

- Static `index.html`, `styles.css`, and `app.js`
- Local storage for tasks, routines, timer preferences, and accessibility preferences
- A calm black cat companion brand direction
- Task brain dump, energy labels, next-step picker, tiny-step breakdowns, focus timer, routines, and overwhelm mode

## Deploy

### GitHub Pages

Upload this folder to a repository and enable GitHub Pages for the branch.

### Vercel

Import the repository in Vercel. No build command is needed because this is a static app.

## Local preview

Open `index.html` in a browser, or serve the folder with any static web server.
